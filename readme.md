# laravelMultiAuthUserManagement
Laravel application to manage user registration and management by admin login

To install the application
1. Navigate to the project root folder.
2. Browse the install url in your favourite browser i.e. http://localhost/project-folder/public/install
3. Follow the step that appear on the screen.
4. Check the depedency required. Recommended php version 7.1+ and laravel version 5.5.x
5. Make the storage/log, storage/framework/ and  bootstrap/cache/ to 775
6. Change default database to sqlite in .env setting file
7. Once the installation get complete browse to public folder to view your application http://localhost/project-folder/public/

To update the application
2. Browse the update url in your favourite browser i.e. http://localhost/project-folder/public/update
